"ProjectSpeech is a STEM non-profit organization that encourages growth in the Computer Science field for those with disabilities. ProjectSpeech promotes voice-based programming for those who cannot use their hands. The project surfaced first in Spring 2017 and officially launched Winter 2017."

Tavis Rudd at Pycon

Travis Rudd introduced a revolutionary program at Pyon last year. Using the popular coding language Python, Rudd was successfully able to code a program that would code on voice command. For example, saying "slap" would make a space and "up 10" would move your code up 10 lines. Check him out!
